This block allows you to add web pages to the workflow. The agent will visit each page from this list. You can add a single page, a list from the clipboard, generate links to numbered pages, and import a list of URLs from a CSV file. Adding the next step 'Follow links' is not required.

Clicking on a site added to the list allows you to specify interaction command for the agent with the page.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------The 'Add a numbered range of pages' option in the extended add menu allows you to automatically generate links to individual pages of the website. This will enable multiple agents to open numbered pages simultaneously (if free agents are available), as opposed to sequentially opening each page by one agent with the 'Paginate through a list of items' block.\

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------The Open Websites block also supports the use of variables. \
Variables are added using the "Apply Input Variables" block. The column name is the name of the variable, and each row below contains the variable value, which will be substituted in all supported blocks where the variable name is referenced.
